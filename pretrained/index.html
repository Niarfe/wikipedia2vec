Pretrained Embeddings

We provide the embeddings in binary and text format. The binary files can be loaded using the Wikipedia2Vec.load() method (see the basic usage). The text files are compatible with the format of the Word2vec. Therefore, these files can be loaded using other libraries such as Gensim's load_word2vec_format(). In the text files, all entities have a prefix ENTITY/ to distinguish them from words.
